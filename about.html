About The Website

This website is about a surreal survivor-horror game from Poland "Darkwood". This website is created by Aktan Zhakypbekov as his first project in web development created using front-end web technologies such as HTML, CSS, and Javascript. The main content of the website is divided into four categories: 1.Game, 2.Plot, 3.Characters, 4.Inventory. 1.Game part gives essential information about the game, its history, gameplay, its creators, and other content. 2.Plot part summarizes the main scenario of the game. 3.Characters part contains information about the RPG aspect of the game. It gives information about the main character, all the secondary characters, and all the units in the game; 4.Inventory part contains information about all the inventory items in the game.
